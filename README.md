# Planner

A day planner for my personal use

Features: 

 - Todo list
 - Event scheduler
 - Markdown based Note taking system with PDF export capabilities
 - Users can save/load their markdown files
 - Iteractive calendar

How to run it? 
```{sh}
python3 planner.py
```
